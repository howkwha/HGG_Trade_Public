# btcusdt_full_history_sqlite_hgg_view

SQLite-exported HGG_VIEW package for local chart-review tooling validation.

- Symbol: `BTCUSDT`
- Timeframes: `4h, 1d, 1w`
- Mode: `full_history_sqlite_view`
- Source: local SQLite chart store

Open `index.html` in a browser to review the bundled chart data.

This package is not a signal, backtest, automation, order/API, or live-trading tool.
