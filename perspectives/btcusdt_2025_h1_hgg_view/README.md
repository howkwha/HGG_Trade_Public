# BTCUSDT 2025 H1 HGG_VIEW Package

Open `index.html` in a browser.

This package contains BTCUSDT public Binance USD-M futures candles for:

- 4h
- 1d
- 1w

The visible window is `2025-01-01 00:00 UTC` through candles closed before `2025-07-01 00:00 UTC`.

The daily perspective and Korean WAL report are in:

```text
daily_perspective_ko.md
```

This is an interpretive practice artifact, not a trading signal, backtest, order/API tool, or confirmed strategy rule.
